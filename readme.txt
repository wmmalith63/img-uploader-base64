== Description ==

this tool help to upload your images easly and after it will convert into base 64 encode and store in a  image tag(<img src=""/>) 
also belove the image that image base64 encode also you can see 

== Installation ==

1. Upload `img-uploader-base64.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php do_action('img-uploader-base64'); ?>` in your templates
